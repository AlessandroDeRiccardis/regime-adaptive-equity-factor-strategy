# Legacy Materials

This directory preserves the original coursework implementation and report for
traceability. It is not the public-facing research pipeline.

- `original_notebooks/`: original monolithic notebook, including outputs.
- `original_reports/`: original PDF, methodology document, and generated figures.
- `original_scripts/`: reserved for original scripts; none were present.

The original files contain assignment wording and several metric/benchmark
descriptions corrected in the refactored repository. They are preserved unchanged.
